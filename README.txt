*Cách cài đặt game Car Racing và chạy.
B1: Hãy truy cập vào đường link trên: https://github.com/TuNgocAn/BTL-nh-m-11-8.
B2: Tải file zip game trên bang cách nhấn vào biểu tượng <>Code, nhấn vào và tiếp tục chọn Download ZIP.
B3: Giải nén file ZIP trên sẽ ra một file, nhấn vào đó và tìm file x64/Debug.
B4: Lúc này bạn sẽ tìm thấy một file CarRacing.exe. Bạn hãy ấn vào để chạy game.
